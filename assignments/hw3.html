QuickWordList: here
Aim

This homework will get you more familiar with table views and data sources.
Task

Make the following changes to QuickWordList:

  • Disallow rearranging of the rows.
  • Make two sections in the table view. The one you add must not be editable: the user should not be able to delete its rows. You can make the text of the rows in the new section whatever you want, but make at least one row. You can put the new section above or below the existing one.
  • Give the section with the spelled-out numbers a header. Have a look at the methods in UITableViewDataSource for this.
Hints

Like in the first homework, the documentation should be the first place you go for help. There are reference documents for table views and data sources as well as more introductory articles about how table views work.

You’ll need to start paying attention to the section of the index path (indexPath.section) that is passed into some of the data source methods.

You don’t have to touch QuickWordListAppDelegate.
